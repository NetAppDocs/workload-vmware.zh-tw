=== 引入對 Amazon Elastic VMWare Service 的遷移顧問支持

BlueXP Workload Factory for VMware 現在支援 Amazon Elastic VMware Service。您可以使用遷移顧問將本地 VMware 工作負載快速遷移到 Amazon Elastic VMware Service，從而優化成本並更好地控制您的 VMware 環境，而無需重構或重新平台化您的應用程式。

https://docs.netapp.com/us-en/workload-vmware/launch-migration-advisor-evs-manual.html["使用遷移顧問為 Amazon EVS 建立部署計劃"]
